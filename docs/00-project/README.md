# プロジェクト概要

## プロジェクト名

expo-app-starter

## 目的

Expo・React Native・TypeScriptを使うアプリを素早く作り始めるための、再利用可能な共通スターターを作る。

## 基本方針

- フロント画面とバックエンド接続の土台を含める
- 新しいアプリへコピーして使用する
- コピー後は各アプリで独立して管理する
- 小さなUI部品集ではなく、動作するアプリ基盤と完成済み画面を中心にする
- 外部UIライブラリとの境界には、必要最小限の薄いラッパーを設ける
- Finance Appで実際に使い、再利用性を検証しながら育てる
