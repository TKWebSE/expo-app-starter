## 2026-07-24: V3 プロフィール表示・法務導線・登録時メール確認

- 設定画面に表示名を常時表示し、プロフィール編集の結果を確認可能にする。
- 利用規約とプライバシーポリシーを認証不要のアプリ内ページとして提供する。
- 新規登録直後は認証済みにせず、メール確認を必須とする。
- ローカル環境と接続先の設定差異に備え、クライアント側でも登録直後のセッションを破棄する。
- 法務文面は実装用のひな形とし、公開前に専門家を含むレビューを行う。
